LTO 3

Laboratorio scolastico

In breve

Il laboratorio scolastico LTO3 è un progetto di laboratorio d'informatica della scuola secondaria di secondo grado IISS Carlo Emilio Gadda, che si propone di fornire un ambiente di lavoro per gli studenti, in grado di soddisfare le esigenze di apprendimento e di sviluppo delle competenze informatiche.

A proposito di questa pagina

Questa pagina è hostata su un istanza VPS AWS, con un sistema operativo Debian 11, e un server web NGINX. Il sito è stato sviluppato con l'ausilio di Bootstrap, un framework per lo sviluppo di siti web e applicazioni web.
